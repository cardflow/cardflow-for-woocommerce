WordPress plugin for accepting Cardflow gift cards in your WooCommerce webshop.

== Description ==

With the Cardflow plugin for WooCommerce you can accept Cardflow gift cards in your WooCommerce webshop. Customers can enter their card number in the WooCommerce coupon field. Once the card number has been validated, the amount will be processed as a discount.
